第１１章　TIME ATTACK #

● 取得ＧＭ評價

如果能將遊戲全破，就將最高的ＧＭ評價做為攻略目標吧。
達成ＧＭ的最低條件是在１３分３０秒以内取得Ｓ９（120,000分以上）並且全破遊戲。
途中還有一些條件例如在Ｌｖ５００時取得Ｓ４，基本上疊得夠快地形又漂亮都不算太難。

乍聽之下疊得快跟疊得漂亮有點矛盾，但不要忘記了「漂亮的地形能使堆疊更有效率」。
地形漂亮，方塊就好放，也更好掌握地形變化，當然也更容易操作。

一旦能取得ＧＭ，就嘗試在十分鐘以內破關吧。
拼命地手動鎖定方塊應該能讓你闖入十一分鐘，然而從這裡開始，你就不得不為了省下那０．１秒而去精簡你的操作。
從開始到結束共計會有７００多個方塊掉落，每個方塊平均１個縮短０．１秒堆疊的話，整體時間就能省下一分鐘以上。

操作最佳化１ #

在２０Ｇ的環境，Ｓ要怎麼於這種地形下向右留一行立起來？
慢速的時候還可以蓄力右移，接著倒退一格，然而在２０Ｇ它早就落地了。
一次往右扳一格的話，還必須連續操作四次，太花時間了。

接著、
「一出現就蓄滿力往邊緣飛，碰到牆壁放開搖桿按下旋轉」
這種操作就非常有效率。

同樣的操作可以應用在Ｔ、Ｌ跟Ｉ。
剛開始時要一步一腳印地練習。一旦習慣了，就能以流暢的方式放到定點。

在遊戲初期，即使速度很慢也不用放開搖桿，
「橫移到底時往下加速並順勢旋轉」
這是可以做的事情。

圖例為往左移動的套路。

承上，
「蓄力橫移到邊緣後，倒退一格往下加速並旋轉」
這也是個套路。
如果在地形很高的時候都能自信滿滿地使出這招，五分鐘內突破Ｌｖ５００絕對不是夢。

透過這樣的操作，可以在速度不快的遊戲序盤讓水平輸入次數在兩次以內。
若還存在著像右右右、蓄力右左左這樣的操作，表示操作沒有最佳化。

一次性地消行 #

當消行產生時，ＡＲＥ會比沒消行時多了０．７秒，所以減少消行次數可以縮時間是理所當然。
假設四消的時間損耗為０，代表著每一次的三消會慢０．１８秒、兩消慢０．３５秒，消單行會慢到０．５３秒。

TIME ATTACK時，用Ｉ消除單行來快速復活是不錯的點子。
根據情況，也可能發展成兩消、三消甚至四消。

但是也不能太超過。特別是目標放在從１０分前半突入９分後半的場合，
「不是無腦消行而是維持堆疊的節奏」顯得更為重要。
例如在ARIKA的官方網頁中發佈的TGM影片裡，雖然也是以四消為主，但也看得見不少其他消行夾雜其中。儘管如此，透過操作最佳化和維持堆疊的節奏，還是能取得９分１９秒的超快成績。
在不迷失自己的前提下慢慢地提升技術吧。

這種情況下來了S，又找不到合適的地點放該怎麼辦？
如果是２０Ｇ，還是有方法將它擺好，發生在遊戲初期的話反而會浪費時間。（然而當落下加速到一定程度，也不用那麼刻意去避開它了。）

如果直接往最右邊插是可以消一行，但還要再消一行才有辦法復活。
後續地形也可能因為立起來的方塊而變得險峻。

建議的做法是讓它橫躺並直接蓋上去。
這樣就可以只消行一次就成功復活。

此外，再上方依序疊上「Ｏ→Ｌ」就可以等著做四消復活了。把這個套路學起來。
Ｌ的放法對於「在狹窄的地面上多蓋一層」這點相當有用。

Ｚ就像這樣立起來吧。
上面再加個Ｔ就可以發展成三消復活。

開場就疊得高高的 #

雖然手動加速有１Ｇ這麼快，但離地面太遠還是會損失很多時間，因此開場就疊高以減少方塊的滯空時間，這是標準套路。 目標是９分多ＧＭ的場合，你應該要能在１６層左右的高度，不影響堆疊的前提做最佳化操作。

就算疊得再高，只要有留最右側兩行就可以削，所以不太可能因為配牌不好而導致窒息。
如何縮短落下距離是勝負的關鍵，所以最好優先處理左側，儘可能地將高度疊到頂端。

另外，考量到手動加速落地的時間需要０．３秒，而三消則只要花０．１８秒，
「頂端的三消比底部的四消來得更有價值」
也就是說，只要能做出同時聽牌三消跟四消的地形，就能堆疊得更穩健，也減少猶豫所帶來的損失。

如果在低處形成了空洞，不第一時間考慮復活，而是在上方持續堆疊通常會比較好。
遊戲的序盤是地勢高、落地快的人勝利，因此復活的考量應該在開始加速之後才進行。

「這樣的妥協消行方式，稱不上是失誤」

值得注意的是隨著地形增高，方塊出現到鎖定的時間會逐漸縮短，整體的節奏也必然越來越快。
如果還繼續維持剛開始的手感，會導致你的反應變慢，連帶影響到蓄力橫移慢半拍，最終造成定位上的失誤。

操作最佳化２ #

配合Ｃ鈕快速使出二連轉的「１８０」。
利用踢牆的特性來製造連續使用的機會。
像這種情況，使用IRS預先C轉，出現的瞬間同時輸入A跟下，這樣的操作是可以實現最速鎖定的。

順帶一提，第７章曾經提過，１frame 的處理優先度依序是「旋轉→水平移動→落下」，
手動鎖定的判定則是接續在這之後。

因此在這種情況下，第二次旋轉跟手動鎖定是可以同時（在同一 frame 中）進行的。

也可以在左側的「牆壁補土」時使用１８０的技巧。
這時候「讓無名指（我個人手小了點所以使用小指）比食指稍微彎一點，利用手腕的重量刷鍵（依照C→A的順序）就可簡單達成，利用下面的思維在腦內整合成單一動作。
直接理解為「想用１８０就是刷鍵」而不是「按兩次Ａ＝ＡＣ」會更加簡單。

將Ｚ「左１」立好之後，Ｊ輸入「右B左」。
妥善使用踢牆能免去逐格掉落，是攻略TIME ATTACK與TAP高難度關卡的強力武器。

在踢牆之前，別忘了要將方塊橫移一格。
基於「旋轉→水平移動→落下」的法則，就算按了旋轉，如果跟橫移卡在同一 frame 裡也會優先旋轉導致放歪，是個小問題。
還不習慣的話，建議先一步一步慢慢來。

→

立用踢牆達成只需往右動一格的例子。
這也讓第二次的旋轉與手動鎖定可以同時發生，無論是「蓄力橫移→下」或「移一格→下」都能夠在同樣的時間點鎖定方塊。
利用踢牆最佳化操作的優勢在於，不僅減少操作的時間成本，還能「維持手動鎖定的節奏」。反之用逐格滑落的方式擺方塊，還要花時間去調整節奏。

儘管使用機會不多，「臥倒」也可以用來最佳化操作。

先蓄力再說 #

「先蓄力再說」指的是與其一邊考慮擺放才一邊蓄力，總之都先蓄力拉到旁邊再思考堆疊吧。
在TGM中，指的主要是挪往左側的蓄力橫移。

TIME ATTACK中理想的操作是「方塊出現時就已完成蓄力，移到邊緣後連１frame 都不浪費馬上鎖定」，實際上能做到多完美是個問題，蓄力太晚所帶來的微小損失，積少成多地變成了對TIME ATTACK的干擾。

利用「先蓄力再說」的技巧在左側進行堆疊，可以有效地將蓄力慢半拍的風險降低至0。就算被迫做一些放棄蓄力的操作，也不至於浪費太多時間。
如果你能參考當下地形與抽牌的趨勢，知道何時該用這招，效率將會大大地提升。
最好是能夠連續來個三四塊，搖桿一直維持著「蓄力左移→方塊出現瞬間手動鎖定，馬上再蓄力左移→……」的循環操作。

活用「先蓄力再說」的技巧也有助於縮短手動鎖定時間。
TGM的方塊是有重量感的（需要較多的蓄力時間），因此搞不清楚「我的蓄力移動到底有沒有慢了」的情況還蠻常發生的。這也使得我們很難在理想的時間點輸入蓄力指令。
可是一旦「先蓄力再說」的技巧成功了，就能夠確定自己的蓄力操作沒有延遲。

「這塊放置的時間點很完美，來處理下一塊吧」像這樣透過不斷嘗試，找出最完美的時間點也逐漸變得容易。

雖然有著各式各樣的優點，用過頭了還是可能招致地形崩毀，要特別注意。
